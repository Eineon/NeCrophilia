# 死亡本能：游戏规则

## 底层规则

### 缩写的含义

- **GM (Game Master)**：游戏的主持人。
- **PL (Player)**：游戏的玩家。
- **PC (Player Character)**：由PL操控与扮演的角色。
- **NPC (Non-Player Character)**：不由PL操控与扮演的角色。
- **N (Normal)**：稀有度，常见。
- **R (Rare)**：稀有度，罕见。
- **S (Special)**：稀有度，稀有。
- **U (Unique)**：稀有度，独特。
- **LV (Level)**：等级，特指角色的人物等级。
- **XP (Experience Points)**：经验值。
- **HP (Hit Points)**：结构所受的损伤。
- **SP (Stamina Points)**：精力所受的损伤。
- **PB (Proficiency Bonus)**：熟练加值。
- **DC (Dormant Class)**：被动检定的判定值。
- **DP (Damage Points)**：伤害的威力。
- **DR (Damage Reduction)**：对伤害的减免。
- **BS (Bad Status)**：基础减益状态。
- **GS (Good Status)**：基础增益状态。
- **AU (Automatic)**：自动动作。
- **ST (Standard)**：主要动作。
- **MI (Minor)**：次要动作。
- **TO (Total)**：全力动作。
- **FR (Free)**：自由动作。
- **RA (Rapid)**：即时动作。

### 括弧的含义

| 类型  | 内容               |
| :---: | ------------------ |
| 「」  | 具有特殊含义的名词 |
| 【】  | 动作或器官的名称   |
| 《》  | 领域的名称         |
|  <>   | 状态的名称         |

### 小数的处理

&emsp;如果某一次计算的结果得出小数，若无特别说明，默认将之往绝对值减小的方向舍去。如果一次计算的过程中会产生小数，则在计算结束之后再进行舍去。  
&emsp;以分数或百分数的形式存在的小数，不会进行舍去，而是将之保留。不过百分数的千分位，以及更小的分位依然会被舍去。

### 稀有度
&emsp;一些元素拥有稀有度，以表示其常见程度。稀有度主要适用于能力和道具。稀有度分为4个级别，按泛滥到稀少的顺序排列：N、R、S、U。  
&emsp;元素的稀有度并非恒定不变，GM可以根据故事情节修改某一元素的稀有度。
- **N**：这一级别的稀有度非常普遍。未标注稀有度的元素，稀有度默认为N。
- **R**：这一级别的稀有度很难获取。不过角色只要足够努力，最终依然可以获得，这可能需要消耗大量的时间。
- **S**：这一级别的稀有度几乎不可能被找到。只有在GM许可的情况下才能获得。
- **U**：这一级别的稀有度通常独一无二。每一个「真名」的稀有度都是U。

## 检定

*当不确定角色的举动是否成功时，会需要进行一次检定。检定分为2种：主动检定与被动检定。无论哪种检定都遵循相同的一套基本流程。*

- **第一步：计算骰值**
  + 投掷2D10，并将出目相加，得到的结果便是骰值。
  + 如果是被动检定，则直接将10作为骰值，不需要进行掷骰。被动检定不受改变出目的效果影响。
- **第二步：计算判定值**
  + 将与检定相关的所有修正值作用在骰值上，得到的结果便是判定值。
  + 这些修正值的总合，被称为「总修正」。
- **第三步：计算达成度**
  + 检定的达成度 = 判定值 - 目标值。
  + 如果达成度超过0，则造成「成功效果」。如果低于0，则造成「失败效果」。达成度为0时，由GM决定造成哪方的效果。GM也可以选择不造成任何效果，或只造成一部分成功或失败效果。
  + 达成度到达10时，可以选择造成「大成功效果」(若有)。达成度到达-10时，必须选择造成「大失败效果」(若有)。
  + 若无特别说明，造成大成功 (大失败) 效果时，不会再造成普通成功 (普通失败) 效果。若无特别说明，在判断其他条件时，大成功 (大失败) 依然被视为普通成功 (普通失败)。
  + 没有目标值的检定，不会拥有达成度，也不会受到与目标值和达成度相关的效果。
  <hr>
- **DC**：被动检定的判定值简称为DC。例如「运动DC」，即是指被动运动检定的判定值。
- **放弃检定**：角色可以自愿放弃自身进行的主动检定。此时跳过掷骰，直接将骰值视为2来判断效果。

### 判定值
&emsp;判定值的大小基于检定的骰值与修正值。以下项目会对判定值的数值产生影响。

#### 修正值
&emsp;加值与减值皆属于修正值。加值与减值总是为自然数，不会低于0，而未说明属于加值还是减值的修正 (例如属性修正) 则可以是负数。  
&emsp;检定获得的加值会与其他修正值相加，减值则会与其他修正值相减。未说明属于加值还是减值的修正，在为自然数时，视为加值处理；在为负数时，将其绝对值视为减值处理。

#### 优势与劣势
&emsp;具有「优势或劣势 [强度]」的检定，在计算骰值时需要投掷额外的D10，「强度」的部分即为额外投掷的D10枚数。例如，具有「优势 2」的检定，需要投掷4D10。

- 进行具有「优势 [强度]」的检定时，投掷 [强度+2]D10，并取其中最高的2枚D10相加作为检定的骰值。
- 进行具有「劣势 [强度]」的检定时，投掷 [强度+2]D10，并取其中最低的2枚D10相加作为检定的骰值。

&emsp;例如，在进行具有「优势 1」的检定时，如果掷骰结果为 [10,5,1]，则将 [10+5] 作为检定的骰值。反之，如果是在具有「劣势 1」的检定中掷出相同的结果，则将 [5+1] 作为检定的骰值。

1. 如果一次检定同时具有多个优势或多个劣势，则仅有强度最高的优势和劣势会产生效果。  
   例如，同时具有「劣势 2」与「劣势 1」，或具有2个「劣势 2」的检定，只会受到1个「劣势 2」的影响，其他的「劣势」不会影响这次检定。
   <hr>
2. 如果一次检定同时具有优势与劣势，则将较高一方的强度减去较低一方的强度，强度较低的那方不会产生效果。如果强度相同，则双方都不会产生效果。  
   例如，同时具有「优势 2」与「劣势 1」的检定，只会受到「优势 1」的效果。
   <hr>
&emsp;当一次检定同时发生上述情况1与情况2时，先进行情况1的处理，随后再进行情况2的处理。  
&emsp;如果角色能够改变或重掷用于计算骰值的D10，这些效果将在优势和劣势的处理结束后发生。  
&emsp;在被动检定中，优势与劣势会被视为 [强度*2] 点作用于骰值的加值与减值，并且依然遵循原本的叠加规则。这些加值与减值，最高不会使骰值超过20，最低不会使骰值低于2。

#### 支援与妨碍
&emsp;有时检定会获得数枚D10的支援骰或妨碍骰，表述为「支援或妨碍 [枚数]」。  
&emsp;在计算检定的修正值时，每一枚支援骰与妨碍骰都会互相抵消，随后进行检定的角色需要投掷剩余的骰子，并从中选出1枚骰值最高的骰子。如果它是支援骰，则这次检定将获得等于骰值的加值；如果它是妨碍骰，则这次检定将受到等于骰值的减值。  
&emsp;多个支援和妨碍效果可以互相叠加，不过最终依然只能选出1枚骰子作为加值或减值。

### 目标值
&emsp;检定的结果基于目标值与判定值之间的差值。目标值有时是一个固定的值，有时会将其他检定的判定值作为目标值。

#### 简易目标值
&emsp;有时GM需要临时决定某次检定的目标值。最简单的方式是先评估这次检定最符合哪个级别的熟练度，随后将 `10+熟练度*10` 作为基础的目标值。

#### 调整目标值
&emsp;GM可以根据检定的难度调整目标值。一些效果也会改变检定的难度。检定的难度，由低到高分为7个级别：-3-极易、-2-轻易、-1-简易、0-标准、1-困难、2-艰难、3-极难。
| 难度级别 |  调整  |
| :------: | :---: |
|   极易   |  -8   |
|   轻易   |  -4   |
|   简易   |  -2   |
|   标准   |  ±0   |
|   困难   |  +2   |
|   艰难   |  +4   |
|   极难   |  +8   |

### 技能检定
&emsp;角色在使用动作时，会频繁地运用技能进行检定，这种检定简称为技能检定。除了关联属性的属性修正之外，技能检定还会获得以下修正。
- **熟练加值**：角色使用熟练度到达1级的技能进行检定时，会获得 `熟练度*2` 点加值，简称为熟练加值 (PB)。
- **等级加值**：角色使用熟练度到达1级的技能进行检定时，会获得 `LV` 点加值，简称为等级加值。

#### 使用技能
&emsp;PL只能告诉GM自己的角色做了什么，而不能直接使用技能进行检定。GM需要根据PL的描述，来判断角色采取的行为会用到的技能。  
&emsp;当GM认为角色的行为需要通过检定确定结果时，才会由GM要求PC进行相应的技能检定。

#### 代偿技能
&emsp;进行技能检定时，角色可以将属于同一类型，且熟练度到达2级的其他技能作为代偿技能使用。这会将检定所用的技能改为代偿技能，但这次检定需要承受8点减值。

### 攻击检定
&emsp;当角色执行 `攻击` 动作时，会频繁的使用攻击技能进行检定，简称为攻击检定。  
&emsp;当角色进行攻击检定时，需要使用相应的攻击技能进行1次技能检定。使用的攻击技能则基于攻击的类型。攻击检定的判定值简称为ATK，达成度简称为POW。  
&emsp;攻击的类型分为「近战攻击」与「远程攻击」两种。攻击的模式也分为「武器攻击」与「魔法攻击」两种。进行攻击检定时，使用的熟练度基于攻击的模式，而非技能原本的熟练度。

- **近战攻击**：近战攻击的攻击技能默认为「运动」。这类攻击是指对处于角色的近接范围内的对象作为目标发起的攻击。
- **远程攻击**：远程攻击的攻击技能默认为「灵巧」。这类攻击是指对近接范围之外的对象作为目标发起的攻击。不过也存在能够进行近距离攻击的远程攻击。
  <hr>
- **武器攻击**：武器攻击的熟练度基于角色对所用武器种类的熟练度。这类攻击是指使用诸如肉体、武器等物体发起的攻击。
- **魔法攻击**：魔法攻击的熟练度基于角色的魔法源流。这类攻击是指通过超自然能力发起的攻击。
  <hr>
- **近接范围**：角色的近接范围是一个半径 [近接距离] 格的球形区域。近接距离基础为1。
  <hr>

#### 连击减值
&emsp;角色在她自己的两个回合之间，多次因执行 `攻击` 动作而进行攻击检定的场合，将会承受一定额度的减值。这一减值简称为连击减值。  
&emsp;角色因执行 `攻击` 动作而进行攻击检定的场合，她会在这次检定结束后承受4点连击减值。之后，每当她因执行 `攻击` 动作而进行攻击检定时，ATK将会受到她当前承受的所有连击减值。  
&emsp;与其他减值不同，连击减值最多只会与其自身叠加1次，较早的减值会被覆盖。每当角色开始回合时，她承受的连击减值会变为0点。  
&emsp;例如，角色在1回合内连续执行4次【动武】的场合，她的第1次【动武】的ATK不会承受连击减值，第2次【动武】的ATK会承受4点连击减值，第3次与第4次【动武】的ATK会承受8点连击减值。

### 防御检定
&emsp;当角色被迫遭受某种效应时，有时会需要使用防御技能进行技能检定，以决定那个效应对角色的作用幅度。这一技能检定简称为防御检定，或是防御。  
&emsp;进行防御检定时，使用的防御技能基于防御的类型类型。防御检定的判定值简称为DEF。DEF会成为判断攻击检定成功与否的目标值。  
&emsp;防御的类型分为「闪避」、「强韧」、「反射」与「意志」四种类型。默认将「闪避」作为应对 `攻击` 动作的防御类型。进行防御检定时，将使用角色对防御类型的熟练度，而非技能原本的熟练度。
- **闪避防御**：闪避防御的防御技能默认为「灵巧」。这类防御代表躲避袭来的物体的能力。
- **强韧防御**：强韧防御的防御技能默认为「耐力」。这类防御代表对损害肉体的效果的承受力。
- **反射防御**：反射防御的防御技能默认为「洞悉」。这类防御代表应对突发情状的处理能力。
- **意志防御**：意志防御的防御技能默认为「直觉」。这类防御代表对抗影响心灵与精神攻击效果的能力。

## 伤害

*伤害会根据其类型，对目标造成结构损伤或精力损伤。*

### 结构损伤
&emsp;结构损伤 (HP) 代表事物的结构所遭受的损害。所有角色，以及一部分物景会拥有结构损伤。结构的损伤阈值基础为4，上限等于阈值的2倍。  
&emsp;角色的每个部位都拥有独立的HP，每个HP都对应着4个不同级别的负伤区间。当部位的HP处于某个负伤区间之内时，该部位便会陷入相应的负伤级别。  

- **0级-无伤**：HP为0的部位处于<无伤>。  
  <无伤>状态本身不具任何效果，不过一部分动作会将部位是否<无伤>作为条件。
  <hr>
- **1级-受伤**：HP超过0，但低于阈值的部位处于<受伤>。
  <受伤>状态本身不具任何效果，不过一部分动作会将部位是否<受伤>作为条件。
  <hr>
- **2级-重伤**：HP到达阈值，但低于上限的部位处于<重伤>。
  <重伤>的部位会在以下事件发生时增加1点HP。同一部位每刻最多只会因此增加1点HP。
  + 使用该部位执行一次性产生2点或更多AP的动作。
  + 使用该部位执行 `操作` 、 `移动` 或 `攻击` 动作。
  <hr>
- **3级-欠损**：HP到达上限的部位处于<欠损>。  
  <欠损>代表角色失去了这个部位。对其他效果而言，视<欠损>的部位并不存在。所有部位全部陷入<欠损>的角色会同时陷入<解体>。
  <hr>

> ▼ **改变负伤状态**  
｜一部分效果能够直接改变负伤状态。遭受这种效果的角色，如果之前的HP低于 `负伤区间的最低HP要求`，则她的HP会升至该数值。  
｜反之，如果她之前的HP高于 `负伤区间的最高HP要求`，则她的HP也会降至该数值。  
｜由此导致的HP变化不属于「伤害」的范畴。

#### 崩解检定
&emsp;要害部位处于<重伤>时，角色在每轮开始时必须进行1次目标值为 `10+LV*2` 的「强韧防御」。如果在<昏迷>中进行这一检定，将受到「妨碍 1」。  
&emsp;要害部位处于<欠损>时，角色会立即<解体>。
> - **大成功**：直到一幕结束，或再次受到伤害为止，角色无需再进行崩解检定。
> - **失败**：角色陷入<昏迷>，直到<负伤>降至1级或更低。
> - **大失败**：如同失败，该部位还会受到「失血伤害 1」。

> ▼ **龙套的结构值**  
｜`龙套` 角色的所有部位共用同一个结构值。如果她不同部位的结构值上限不同，则将最高的结构值作为共用的结构值。

### 精力损伤
&emsp;精力损伤 (SP) 代表角色的意志力与体力所遭受的综合损耗。精力的损伤阈值为 `耐力PB+CON+FTH (最低2)`，上限等于阈值的2倍。完成短休可以消除1点SP。完成长休则可以消除最多等于损伤阈值的SP。  
&emsp;SP到达阈值的1/2的角色会陷入<疲倦>。SP到达阈值时，角色还会陷入<力竭>。角色因其他原因陷入<力竭>时，如果SP低于阈值，则会增至阈值。当SP到达上限时，角色会立即<解体>。  

> ▼ **透支检定**  
｜如果角色处于<力竭>，则她的SP每次增加后，都必须进行1次目标值为 `10+(当前SP*2)` 的「强韧防御」。如果在<昏迷>中进行这一检定，将受到「妨碍 1」。
> - **失败**：角色陷入<昏迷>，直到SP低于阈值。
> - **大失败**：如同失败，角色的「核心部位」还会受到「毒素伤害 1」。

### 伤害类型
&emsp;所有类型的伤害一共分为「物理伤害」与「精神伤害」两类。若无特别说明，物理伤害只会增加HP，精神伤害只会增加SP。

> ▼ **物理伤害**  
｜<span class=V>v</span> 冲击：由物体运动产生的力量粉碎目标，如撞击、震荡等。  
｜<span class=V>v</span> 切割：利用锋利的物体分割目标，如割伤、裂伤等。  
｜<span class=V>v</span> 穿刺：通过物体尖端或锐利边缘直接刺入目标，如刺伤、穿刺伤等。  
｜<span class=V>v</span> 毒素：因有毒物质进入体内引起的生理机能异常，如毒药、瘴气所致的伤害。  
｜<span class=V>v</span> 失血：由于开放性伤口或其他原因导致的血液损失，如内出血、因外伤导致血流不止。  
｜<span class=V>v</span> 灼烧：通过高温或化学物质造成的烧伤，如火焰、闪电、酸性物质所致的伤害。  
｜<span class=V>v</span> 冷冻：通过极低温度引起的冰冻伤害，如低温环境、液氮所致的伤害。  
｜<span class=V>v</span> 腐蚀：化学物质侵蚀目标物体导致的伤害，如氧化、腐蚀性液体、核辐射所致的伤害。

> ▼ **精神伤害**  
｜<span class=V>v</span> 困顿：长时间持续使用肌肉或精神的过度疲劳，如长时间奔跑、超负荷工作。  
｜<span class=V>v</span> 疼痛：生理和心理上的疼痛感觉，如刺痛、灼痛等。  
｜<span class=V>v</span> 欣快：由积极、愉悦和兴奋的情绪对判断能力产生的影响，如性冲动、致幻药物等。  
｜<span class=V>v</span> 屈服：由心理上的压力、负面情绪、不安和无助感导致的心理退行，如强迫行为、压迫感、恐惧等。 

#### 复合伤害
&emsp;有时一次伤害拥有不止1种伤害类型，这类伤害被称为「复合伤害」。  
&emsp;如果一次复合伤害同时具有物理伤害与精神伤害，则它造成的伤害会同时增加HP与SP。

### 造成伤害
&emsp;从伤害产生，到对象受到损伤，共分为以下2个步骤。
- **第一步：计算DP**
  + DP代表伤害的威力，基础为「[伤害类型]伤害 X」中的X，再加上其他改变DP的效果。
  + 如果DP来自 `攻击`，此时将会触发与「造成攻击」和「受到攻击」相关的效果。
  + 如果DP来自 `攻击`，且 `POW` 不超过 `对象的DR`，DP将受到2点减值。不存在POW的效果不会受到这个减值。
  + DP最低为0。DP为0的伤害不会进入下一个步骤。
- **第二步：计算DMG**
  + DMG代表对象承受的损伤额度，基础等于DP，再加上其他改变DMG的效果。
  + 此时，将会触发与「造成伤害」和「受到伤害」相关的效果。
  + DMG最低为1。
  <hr>
- **抗性**：计算DP时，如果对象具有对此次伤害的抗性，DP将受到2点减值。
- **易伤**：计算DP时，如果对象具有对此次伤害的易伤，DP将获得2点加值。
- **复合伤害**：单一伤害类型的抗性对复合伤害无效，除非对象拥有对构成复合伤害的所有伤害类型的抗性。而单一伤害类型的易伤依然会对复合伤害起效。

#### 障壁
&emsp;障壁可以代替角色承受损伤，表述为「障壁 [强度]」。若无特别说明，障壁可以承受任何类型的损伤。强度为0的障壁会立即消失。
- DMG会优先1:1地扣除障壁的强度。障壁消失后，剩余的DMG (若有) 会如常增加角色的损伤。
- 被扣除的障壁强度无法恢复，也无法用于承受代价。
- 如果DMG完全由障壁承受，则此次伤害不会触发与「造成伤害」和「受到伤害」相关的效果。
- 未说明持续时间的障壁，默认在一章结束时消失。
- 若无特别说明，不同的障壁不会叠加，角色只能择一保留。

### 特效

> ▼ **切断**  
｜受到「切断」伤害的角色，受伤的部位将陷入<欠损>状态。  
｜对于 `龙套` 角色，「切断」的效果会变为「伤害的DMG获得2点加值」。

> ▼ **爆发**  
｜受到「爆发」伤害的角色，必须选择1个与受伤部位衔接的其他部位，该部位会受到1次DP与该DMG相同的伤害，不会附带任何特效，也不会造成精神伤害 (若为复合伤害，则失去精神伤害类型)。如果受伤部位不与任何部位衔接，则该效果无效。
｜对于 `龙套` 角色，「爆发」的效果会变为「伤害的DMG获得2点加值」。

> ▼ **处决**  
｜只有要害部位可以被选为承受「处决」伤害的部位。受到「处决」伤害的角色，会立即<解体>。  
｜如果受伤的角色不具要害部位，则「处决」无效。  
｜对于 `龙套` 角色，「处决」的效果会变为「受到这个伤害的角色，将立即<解体>」。无论是否具有要害部位，「处决」都会对 `龙套` 生效。

> ▼ **贯通**  
｜受到「贯通」攻击的对象，DR会受到等于DP的减值。  
｜对于 `龙套` 角色，「贯通」的效果会变为「这次攻击无视DR」。

> ▼ **故障**  
｜受到「故障」伤害的角色，受伤部位将受到「创伤\故障：无法使用具有 `操作` 或 `攻击` 标签的动作」。  
｜对于 `龙套` 角色，遭受的「创伤\故障」的效果会变为「无法使用AU之外的动作」。

> ▼ **侵蚀**  
｜受到「侵蚀」伤害的角色，受伤部位将受到「创伤\侵蚀 `伤害类型`：每轮开始时，该部位受到 `相同伤害类型的侵蚀创伤数量` 点伤害」，伤害类型与造成「侵蚀」的伤害相同。

## 效果

*游戏中的所有行为都会产生效果。有些效果很容易判断，比如拿起一把撬棍；而其他情况可能需要更多规则描述才能确定效果，有时候甚至需要进行检定来确定效果的强度。以下是关于效果的通用规则。*

### 条件
&emsp;一些效果有着独特的使用条件。如果某一效果的条件涉及角色的某项数值，而角色的对应数值超过了效果的数值条件，若无特别说明，依然视该角色满足该效果的条件。
- **前置**：角色只能取得满足前置条件的效果。当一个已经取得的效果的前置条件不再被满足时，视角色不再具备那个效果，直到前置条件被重新满足。
- **需求**：具有需求条件的效果，只会在满足需求条件的情况下产生效果。虽然即便不满足需求条件也可以使用这些效果，不过这么做只会白白浪费时间，但不会消耗使用次数。
- **触发**：具有触发条件的效果，只能在满足触发条件时使用，并且不能针对同一回触发连续使用多次。

### 执行
&emsp;动作的执行类型会决定效果的生效时机。执行类型分为6种：自动动作 (AU)、主要动作 (ST)、次要动作 (MI)、全力动作 (TO)、自由动作 (FR)、即时动作 (RA)。
- **自动**：AU动作的效果总是处于被使用的状态。除非动作的描述中说明使用者「可以」停下，否则使用者不能停止使用AU动作。属于AU动作的动作会省略「执行」条目。
- **主要**：ST动作只能在角色自己的回合内宣言。使用大部分的ST动作都会产生AP。若无特别说明，ST动作产生的AP最低只能被降至1。
- **次要**：MI动作只能在角色自己的回合内宣言。宣言MI动作后，角色无法再宣言任何MI动作，直到她的下个回合开始。
- **全力**：TO动作只能在角色自己的回合内宣言。部分TO动作会消耗大量的时间，角色可以将其产生的AP拆分为多个回合产生。
- **自由**：FR动作可以在任何时候宣言，无论在角色的回合内，还是回合外。
- **即时**：RA动作与FR动作类似，不过在宣言RA动作后，角色无法再宣言任何RA动作，直到她的回合开始。所有RA动作都具有「触发」条目。
  <hr>
- **宣言上限**：多个动作的「触发：~ 时」条件被同时满足时，角色只能从中选择1个宣言。

### 代价
&emsp;动作的动作决定了使动作生效所产生的消耗。如果角色无法满足动作所需的代价，动作将会使用失败。

#### 额外代价
&emsp;一些效果会产生额外的代价。额外代价不会被计入原本的代价之中，也不会改变效果的作用。但如果无法满足这些代价，动作依然会使用失败。

### 时效
&emsp;效果的持续时间如果超过一瞬间，则会具备「时效」条目。这段时间结束后，效果也会随之结束。如果同一效果对不同目标有着不同的时效，则分别处理每名目标承受的效果时效。时效存在以下几种表现形式。
- **一段时间**：许多效果以1个代表时长的自然数再加上时间单位作为时效。如果代表时长的是一个随机数，则在效果发动时由GM暗骰决定具体的持续时间，使用者不会知晓具体的持续时间。
- **条件**：效果会持续至满足特定条件。如果同时存在多个结束条件，则以先发生的为准。
- **焦点**：效果会持续至下一轮结束，角色可以执行【聚焦】动作延长持续时间。
- **永久**：效果会一直持续，直到被解除。

### 射程
&emsp;射程代表源点 (即效果的起点) 所能涉及的范围。射程存在以下几种表现形式。没有「射程」条目的效果，以使用者的所在点作为源点。  
&emsp;需要选择目标的效果，即代表将作为目标的对象的所在点，作为效果的源点。若无特别说明，效果的使用者不能将她看不见的点作为源点。
- **自身**：将使用者所处的区域作为效果的源点。这种效果会省略「射程」条目。
- **一段距离**：以2个被「~」相连，代表距离的自然数作为射程。前者表示源点距离使用者的最小距离，后者表示源点距离使用者的最大距离。最小距离与最大距离相同时，会简写为单个数字。若无特别说明，增加射程的效果会同时增加最小射程与最大射程。
- **视界**：使用者可以将她视野内的任何地方作为效果的源点。
- **无限**：使用者可以将任何她已知的地点作为效果的源点，即便她看不见源点。

#### 射程增量
&emsp;部分效果的射程存在增量，表述为「[射程增量]'」。这类效果所能涉及的最大距离为射程增量的6倍，但其强度也会随着距离的增加而逐渐衰减，表现为「衰减度」。  
&emsp;效果的目标距离源点的距离超过射程增量的 [X] 倍的场合，这一效果的衰减度便视为 [X] 点。衰减度到达6点的效果会使用失败。

> ▼ **射程减值**  
｜`攻击` 效果每有1点衰减度，ATK就会受到2点减值。这一减值简称为射程减值。

### 范围
&emsp;发生在一个特定形状的范围内的效果即是范围效果。范围的形状分为4种：球形、扇形、锥形、矩形。这些范围会从源点开始直线向外扩散，影响覆盖范围内的所有区域。  
&emsp;拥有「范围」条目的 `攻击` ，不会因<隐蔽>承受「劣势」。
- **球形**：球形范围以源点区域的任意一个角作为起点，向所有方向扩散到指定的距离后停止。表述为「球形 [半径长度]」。
- **扇形**：扇形范围以源点区域的任意一个角作为起点，扩散为一个半球体。表述为「扇形 [半径长度]」。
- **锥形**：锥形范围以源点区域的任意一个角作为起点，扩散为一个正四面体。表述为「锥形 [高的长度]」。
- **矩形**：矩形范围以源点区域的任意一个角或边作为起点，形成一个与源点区域存在至少1个共边或共角的立方体。表述为「矩形 [长度\*宽度\*高度]」。若没有给出宽度或高度，则将省略的参数视为1。
  <hr>
- **弥漫**：标注「弥漫」的范围能够绕过拐角扩散。表述为「弥漫[范围]」。角色对抗标记有「弥漫」的范围效果时，防御检定不会因物景获得「优势」和加值。

### 自愿
&emsp;一些效果只会对自愿的角色起效。角色的自愿与否由扮演该角色的参与者决定，无关该角色是否保有意识。  
&emsp;若无特别说明，接受这种效果的角色，会立即知晓关于效果的一切信息。

### 效果的叠加
&emsp;若无特别说明，同名能力造成的效果不会叠加。在决定保留哪个不会叠加的效果时，如果属于无害效果，由受到效果的角色决定保留哪个，否则由GM决定保留哪个。通常保留级别更高，且剩余时效更长的效果。 

### 被压制的效果
&emsp;一些效果会在某种事件发生时被压制。被压制的效果不会生效，但其时效依然流逝。

## 感知

### 光照
&emsp;环境的光照阶层影响着角色如何通过眼睛观察事物。光照阶层从强到弱分为4级：强光、常光、薄暗、黑暗。一个常规的光源能够在其所列半径提供的光照之外，在双倍半径的区域中提供更次一级的光照。例如，如果一支火把能够提供照明半径为1格的常光光照，则它还会提供半径为2的薄暗光照。
- **强光**：直视太阳时的亮度。常年生活在黑暗中的生物很容易在这种光照下灼伤眼睛。
- **常光**：散射在空气中的日光亮度。一般的生物可以在这个亮度下通过眼睛清晰视物。
- **薄暗**：日光中投下的阴影，或是月光的亮度。观察处于薄暗中的事物时，视觉的敏锐度会降为「模糊感官」。
- **黑暗**：没有月亮的黑夜中的亮度。观察处于黑暗中的事物时，视觉的敏锐度会降为「疏略感官」。

### 感官
&emsp;生物感知环境的方式基于它所具备的感官。感官的敏锐阶层，可以从强到弱分为3级：精确感官、模糊感官、疏略感官。视觉、听觉和嗅觉是三种最常见的感官，它们有着不同的敏锐阶层。
&emsp;当角色在侦测到其它角色时，通常使用最敏锐的感官。在规则上，视觉是默认的唯一精确感官，听觉是默认的唯一模糊感官，嗅觉是默认的唯一疏略感官。
&emsp;有些生物拥有和默认情况不符的精确与模糊感官。例如，视力不好的角色的视觉是模糊感官；【灵敏嗅觉】能力能够使嗅觉成为模糊感官。
- **精确感官**：视觉默认为精确感官。这类感官能细致地感觉世界的细节。唯一能无缺陷地定位目标的方法就是使用精确感官。
- **模糊感官**：听觉默认为模糊感官。这类感官无法像精确感官一样完整地进行侦测。只能被模糊感官定位的事物，<隐蔽>会升至2级。
- **疏略感官**：嗅觉默认为疏略感官。这类感官只会提醒角色某物是否存在，但完全无法锁定其位置。只能被疏略感官定位的事物，<隐蔽>会升至3级。

#### 特殊感官
- **暗视**：在判断视野内的光照阶层时，拥有此感官的角色可以将「暗视」的级别加在自身感知的光照阶层上，最高不超过常光。例如，拥有「1级暗视」的角色，可以将视野内的薄暗视为常光，将黑暗视为薄暗。角色无法通过「暗视」在黑暗中分辨颜色，只能看到有灰度的黑白画面。若无特别说明，这一感官依赖视觉。
- **振动视域**：角色可以通过接收振动来探查环境，但角色的近接范围内必须存在她能够接触的介质，用于传递振动。角色无法通过「振动视域」感知颜色和对比度。「振动视域」的级别将决定其敏锐度：1级为模糊感官，2级为精确感官。若无特别说明，这一感官依赖听觉。
- **热源视域**：角色可以通过温度变化来探知环境。角色无法通过「热源视域」感知颜色和对比度。并且无法在强度到达3级的炎热或寒冷环境下使用此感官。「热源视域」的级别将决定其敏锐度：1级为模糊感官，2级为精确感官。若无特别说明，这一感官依赖对温度的感知。

### 隐蔽级别
&emsp;<隐蔽>分为4个级别：0级-可见、1级-朦胧、2级-匿迹、3级-无踪。除了光照和感官带来的影响之外，<潜行>和<隐形>状态也可以增加<隐蔽>的级别。<隐蔽>的级别永远相对于目标和观测者。例如，敌人可以对PC<可见>，但是对PC的其他友军<无踪>。
- **可见**：能够被角色用精确感官定位具体位置的所有事物，对角色而言的隐蔽级别都是可见。
- **朦胧**：隐约可见的事物，对角色而言的<隐蔽>级别是朦胧。角色依然可以精准定位朦胧事物的位置，但是以朦胧事物为目标发起的攻击，ATK会受到「劣势 1」。
- **匿迹**：匿迹的事物无法被角色精准定位，只能判断出其处于哪个区域。角色还会对匿迹的事物陷入<破绽：闪避>。此外，角色以匿迹事物为目标发起的攻击，ATK会受到「劣势 2」。
- **无踪**：无踪的事物无法被角色精准定位，就连在哪个区域里都不会知道。角色还会对无踪的事物陷入<破绽：闪避>。角色可以对它怀疑存在无踪事物的区域发起攻击，ATK会受到「劣势 4」，并且不会知晓失手的原因是因为攻击检定失败，还是因为选择的区域内不存在无踪的事物。

## 模式

*游戏规则会在3种模式间切换：探索、冲突、休整。GM会根据故事的流向调整游戏的模式。*

### 时间
&emsp;游戏中经常会用到3种独特的时间单位，分别是轮、幕和章。一章由数个连续的幕构成，而一幕又由数个连续的轮构成。这些单位在游戏中代表的时间一般由剧本决定，也可以由GM自行决定。  
&emsp;每经过1章，通常意味着度过了1天。根据剧本的不同，也可能是1周、1个月，甚至1年。

### 探索模式
&emsp;在探索模式中，角色需要掌握情况，与伙伴交流情报，或是进行长距离移动。在探索模式中度过的幕简称为「常幕」。一些动作具有 `探索` 标签，代表其不会在冲突模式中生效。

### 冲突模式
&emsp;当角色需要讨伐敌军、舌战群儒，或发生其他激烈而短暂的事件时，便会从探索模式进入冲突模式。在冲突模式中度过的幕简称为「战幕」。  
&emsp;在冲突模式中，时间会被分割为「轮」，每轮约等于游戏世界内的1分钟。无论冲突的起因和目的，皆以同一套流程进行。

#### 第一步：决定行动顺序
&emsp;所有参与冲突的角色都需要进行1次先攻检定，随后将该检定的判定值作为先攻值，以先攻值由高到低的顺序开始回合。  
&emsp;若多名互相敌对的角色的先攻值相等，则由先攻检定的修正值较高的角色先开始回合，其次由LV较高的角色先开始回合，最终由GM扮演的角色先开始回合。  
&emsp;其他先攻值相等的角色，可以自行商量互相之间开始回合的顺序。

> ▼ **先攻检定**  
｜使用先攻技能进行技能检定，简称为先攻检定。默认将「洞悉」作为先攻技能。先攻检定不存在目标值。

#### 第二步：回合进程
&emsp;每一轮皆由连续的20个刻度构成。在第20刻，会开始先攻值为20的角色的回合。所有先攻值为20的角色的回合结束后，时间会进入19刻，由先攻值为19的角色开始回合，以此类推。  
&emsp;当第1刻结束时，即代表这一轮结束。随后冲突会进入下一轮，刻度也会变回20。  
&emsp;角色结束自己的回合时，她的先攻值会减少20点。下一轮开始时，所有角色的先攻值都会增加20点。  
&emsp;在角色的回合开始前，她可以使用【延后】，令自己的先攻值-1，以放弃在本刻进行回合。

> ▼ **主要先攻与当前先攻**  
｜如果一名角色的先攻值超过20，她将有机会在同一轮中进行多个回合。先攻值超过20的角色，先攻值会被分为「主要先攻」与「当前先攻」。  
｜「当前先攻」在每轮开始时等于「主要先攻」(最高20)，用于判断角色何时开始回合。原本影响先攻值的效果，只会影响「主要先攻」。  
结束回合时，角色的「主要先攻」不会减少，而「当前先攻」只会减少 `400/主要先攻 (向上取整)` 点。  
｜每轮开始时，角色的「主要先攻」不会增加，而「当前先攻」依然会增加20点。

#### 第三步：结束冲突
&emsp;当引发冲突的事端被解决时，冲突模式也会随之结束。此时GM会发放XP、宠爱或其他奖励。  
&emsp;之后通常会继续进行探索模式，也有可能开始下一个冲突模式。

#### 变体：沉淀先攻
&emsp;若启动此变体，则在每轮结束时，所有先攻检定的骰值低于11的角色，骰值会+1；高于11的角色，骰值会-1；等于11的角色，骰值不会发生变动。

### 刻度
&emsp;刻度又被简称为刻。随着每刻的开始与结束，会自动发生许多事件。每刻可以分为以下3个步骤。

#### 第一步：刻的开始
&emsp;当一刻开始时，会进行以下效果的处理。在同时处理不同角色产生的效果时，处理顺序与她们的行动顺序相同。在处理同一角色产生的效果时，由她自行安排处理顺序。

- 所有以「刻」作为时间单位的事件，时间会经过1刻。  
- 若有角色在此刻开始回合，则可以使用以她「回合开始」作为触发条件的动作。所有以这名角色「回合开始」作为时间单位的事件，时间会经过1回合。  
- 其他特别说明会在一刻开始时发生的事件。

&emsp;当上述效果全部处理完毕后，在此刻开始回合的角色会消除 `AP阈值` 点AP。若此刻是一轮中的第一刻，则所有以「轮」作为时间单位的事件，时间会经过1轮。

#### 第二步：刻的进行
&emsp;在一刻中，进行回合的角色能够使用「ST动作」和「MI动作」。角色的回合会在她的AP到达阈值时结束。她也可以主动提前结束回合，这不需要动作。当没有角色能够在这一刻进行回合时，这一刻也会随之结束。

> ▼ **行动值**  
｜AP是在使用绝大多数动作时都会被增加的一种数值，代表为使效果生效所花费的时间。  
｜AP阈值基础为3，上限等于阈值的2倍。角色开始自己的回合时 (具体见<a href=#刻的开始>刻的开始</a>)，会消除 `AP阈值` 点AP。

> ▼ **拆分全力动作**  
｜使用TO动作时，角色可以将其对AP的增加拆分为多个回合产生，这些回合必须连续，并且每回合至少产生1点AP。  
｜例如，一名角色使用产生9点AP的TO动作时，可以在使用这个动作的回合产生3点AP，随后在她自己的下个和下下个回合分别再产生3点AP，来完成这个动作。动作的效果也会在它被完成时生效。

#### 第三步：刻的结束
&emsp;当一刻结束时，会进行以下效果的处理。在同时处理不同角色产生的效果时，处理顺序与她们的行动顺序相同。在处理同一角色产生的效果时，由她自行安排处理顺序。 

- 结束所有持续至此刻结束的效果。对于在此刻进行了回合的角色，持续至她的回合结束的效果，也会在此时结束。
- 若有角色在此刻进行了回合，则可以使用以她「回合结束」作为触发条件的动作。所有以这名角色「回合结束」作为时间单位的事件，时间会经过1回合。
- 其他特别说明会在一刻结束时发生的事件。

&emsp;若此刻是这轮的最后一刻，还会结束所有持续至这轮结束的效果。处理顺序同上。

### 舞台
&emsp;在冲突模式中，所有参与冲突的角色都会被配置在舞台上。舞台由数个四边形或六边形的格子构成。这些格子被称为「区域」，或简称为「格」。  
&emsp;区域之间的距离以「格」作为单位。这同样也是衡量角色的移动距离，以及效果的射程和范围的单位。区域的面积并不固定，同一区域中可以容纳多名角色，其上限由GM视情况而定。  
&emsp;在计算不同区域之间的距离时，共用一条边的区域，距离为1格。不具共用的边，但具有共用的角的区域，距离为2。

#### 移动
&emsp;角色需要跨越区域的边才能移动进入下一格区域，不能进行对角移动。角色可以连续移动多格，但是每格的选择都必须遵循上述规则。  
&emsp;角色移动到某格时，如果有某种效果会在该处起效，她的移动会被暂停，优先进行这些效果的处理。  
&emsp;当角色遭受会导致「无法移动」的效果时，她的「速度」将变为0，并且使用的任何 `移动` 动作都会被无效。若无特别说明，「无法移动」的角色依然可以通过「特殊移动」进行移动。

#### 速度
&emsp;速度代表角色的基础移动能力。速度基础为 `灵巧熟练度+(STR+DEX)/4 (最低2)`。速度最低只能被降至0。

> ▼ **MOV**  
｜MOV代表角色的当前移动能力。MOV在冲突模式开始时为0点，当角色开始回合时，她的MOV会变为 `速度` 点。她会在使用具有 `移动` 标签的动作进行移动时消耗MOV。  
｜通常境况下，每次进入1个区域时都需要消耗1点MOV。一些特殊的物景，例如崎岖的「地形」，会增加移动进入其中所需要消耗的MOV。  
｜MOV最低为0，角色无法采取产生的消耗会使MOV低于0的移动路线。

> ▼ **打断移动**  
｜部分效果能够打断移动，例如【阻挡】动作。若无特别说明，「强制移动」不会被打断。即便被打断，也只有被作为MOV的移动距离会被降至0，而不会降低被移动的角色的MOV。

> ▼ **作为触发条件的移动**  
｜一些效果会以执行 `移动` 动作作为触发条件，其中的典范便是【截击】。通过 `移动` 动作移动的每一格都能够满足这种触发条件。  
｜满足这种触发条件的是「执行 `移动` 动作」的角色，而非实际进行移动的角色。因此，通过不具 `移动` 标签的效果移动的角色，和因为他人造成的效果而移动的角色，不会满足这种触发条件。

#### 特殊移动
&emsp;除了常规的移动手段，角色还可能因为一些特殊的效果而产生移动，角色在面对这些移动时通常都是被迫的。  
&emsp;特殊移动表述为「[移动类型] [移动距离]」。特殊移动不会消耗被移动的角色的MOV，而是将其中的「移动距离」作为MOV进行消耗。若无特别说明，由造成特殊移动的角色决定移动路线。  
&emsp;若无特别说明，特殊移动进行的每格移动所消耗的MOV，只会受到移动经过的区域内的「物景」影响，而不会被遭受移动的角色所受的效果影响。  
&emsp;如果遭受特殊移动的角色拥有无视「物景」影响的效果，则造成特殊移动的角色，可以使遭受移动的角色无视相应「物景」的影响。  
&emsp;特殊移动不会使角色进入她无法通过的区域，而是停止在最后一个她可以通过的区域中。

> ▼ **位移**  
｜因冲击而被击飞、瞬间爆发极快的移动，都属于位移的范畴。位移的效果与通常移动相同，不过位移可以由不具 `移动` 标签的效果造成。

> ▼ **传送**  
｜因传送进行的移动，每移动1格距离，固定消耗1点MOV，而不会受到其他效果的影响。被传送的角色会直接进入终点区域，无需经过移动路线上的其他区域。

#### 相位移动
&emsp;具有 `相位` 标签的效果造成的移动，可以使角色进入她无法通过的区域。这一移动无视 `地形` 物景，并且角色在被移动的期间内会陷入<失重>。  
&emsp;如果角色在移动结束时处于她无法通过的区域内，则她会被传送至移动路线中最近的她可以通过的区域内，并且所有部位都会受到「冲击伤害 `传送距离`」。

#### 物景
&emsp;物景是对舞台上，除角色之外的一切事物的统称。例如狭窄的地形、危险的气候等。所有物景一共可以分为4类：地形、墙体、道具、空间。

#### 掩护
&emsp;一些物景可以在战斗中为角色提供掩护。掩护分为3个级别：弱掩护、强掩护、全掩护。
- **弱掩护**：角色的闪避防御获得「优势 1」。
- **强掩护**：角色的闪避防御获得「优势 2」。
- **全掩护**：角色的闪避防御获得「优势 4」，并且无法被选定为效果的目标。

### 休整模式
&emsp; 角色可以在休整模式中进行充分的休息、训练技艺、学习新的知识、或仅仅只是玩乐。在休整模式中度过的幕简称为「结幕」。一些动作具有 `休整` 标签，执行不具 `休整` 标签的动作会打断休息。

#### 休息
&emsp;休息共有3种方案：小憩、短休、长休。完成不同的休息方案所需要消耗的时间各不相同。角色在同一时间只能采取1种休息方案。  
&emsp;在休息的场合执行不具 `休整` 标签的动作，将会导致休息被打断，并以已经消耗的时间结算这次休息的效果。此外，在「静养」时维持 `专注` 效果也会打断休息。  
&emsp;角色在同一章内最多只能完成2次静养，多余的休息时间会被视为连续的小憩。
- **小憩**：小憩是一阵短暂的休息时间，角色可以在小憩时进行安静的活动，例如阅读、聊天、进食、护理伤口等。小憩消耗的时间可以忽略不计，不过在计算效果的持续时间时，完成1次小憩视为经过了10轮。
- **静养**：静养是一段适当的休息时间。角色可以在静养时进行轻松的活动，例如睡眠、阅读、聊天、进食等。一次完整的静养需要消耗至少1幕的时间。
