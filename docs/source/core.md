# 追加规则

## 基础规则

- **最低值**：数值最低为0，不会变为负数。
- **截尾取整**：小数直接舍去。例如：3.7取3，-1.2取-1。
- **特例优先**：如果某一机制与**通常**情况矛盾，优先应用特例的机制。
- **同伴**：除自身之外，GM认为与角色站在同一战线的其他角色。
- **幻化**：PL可以自由更改自身机制的名称与表现形式，以契合角色设定以及演出风格。这不会影响实际的游戏效果。

### 生物
所谓的**生物**是指在机制上被定义为生物的实体，无关其是否具有生命。 \
例如，像机壳、亡灵这样无生命的实体，依然属于生物的范畴。随处可见的草木与细菌虽然拥有生命，但在机制上并不被视为生物，而是当作物品来对待。

所有生物都属于以下谱系之一。
| 谱系  | 含义                                                             |
| :---: | :--------------------------------------------------------------- |
|  凡   | 具备心智的天然生命体。涵盖包括人类在内的所有自然动物。           |
|  魔   | 脱离凡人行列的超自然存在，例如幽灵、吸血鬼与恶魔。               |
|  械   | 通过人工技术构建的生命体，通常缺乏情感与欲望，依赖预设逻辑运行。 |

### 掷骰

#### 奖励骰
妳可以在掷骰后，GM宣布后果前，决定是否为这次掷骰赋予那些妳可以主动添加的奖励骰。 \
无论如何，每次掷骰最多只能获得3枚奖励骰。

#### 重掷
当妳进行重掷时，重新投掷每枚D6，并使用新的出目。每枚骰子通常只能重掷1次。奖励骰通常不会被重掷。

### 议程
这一模块取代了核心《暗夜刀锋》中玩家角色卡的经验触发器。 \
议程用于表示妳的动机与接触世界的方式。妳在一开始拥有以下通用议程。
- 展现自己的信念、动机、疯狂、出身或经历。

除此之外，妳还拥有1项独属于自己的议程。参考以下范例创建妳的专属议程。
- 大战一场，或是隐忍不发。
- 展现妳的人性，或是妳与人性的距离。
- 创造性地解决问题，或是采取简单的解决方案。
- 保护妳的人，或是绝不抛弃任何人。
- 展现卓越技艺，或是卸下伪装。
- 遵守命令，或是违抗命令。
- 将工作置于个人需求之前，或是休息一下。
- 揭露世界隐藏或令人不安的真相，或是揭露自己隐藏或令人不安的真相。
- 带头冲锋，或是让别人带头冲锋。
- 以人为本，或是故意伤害。
- 超越宿敌，或是让宿敌超越妳。
- 展示华丽的超凡力量，或是召致灾难。
- 指使别人，或是无私奉献。
- 充实自己，或是赠予贵礼。

会话结束时，妳在这场会话中每触发1项议程，便获得1点经验。若多次触发同一议程，改为获得2点。 \
若祢觉得旧议程不再适用自己的角色，可以在幕间替换议程。


## 牵绊
这一模块取代了核心《暗夜刀锋》中的恶欲，并调整了压力与创伤的机制。

### 建立牵绊
祢随时可以在游戏过程中，向GM申请为妳与特定对象建立牵绊，或是主动遗忘一段牵绊。 \
牵绊的对象可以是角色、组织、地域、甚至理念。但不能是妳自己。 \
妳也不能重复对相同对象建立多段牵绊。即便牵绊对象不复存在，已经建立的牵绊也不会消失。

> 祢的角色最珍视什么？最无法原谅什么？把这些答案变成她的牵绊。 \
> 妳愿意为守护这段牵绊做到什么程度？如果必须伤害它才能活下去，妳会怎么做？

#### 临时牵绊
有时妳会被授予临时的牵绊，代表妳与特定对象的短暂关系。 \
妳通常会在刻痕结束时遗忘所有临时牵绊。祢随时可以向GM申请将临时牵绊转正。

### 情感
妳对待牵绊的情感分为2个侧面：**闪耀**与**罪恶**。 \
使妳承受压力或枯竭的机制改为给予等量的罪恶。 \
妳的**罪阈**决定了妳能承受的罪恶上限，初始为9。

#### 罪恶满溢
罪恶一旦到达罪阈，妳的心智便会被翻涌的恶意压垮，陷入罪恶满溢状态，持续至消除所有罪恶。 \
妳会在陷入罪恶满溢时犯下1种罪行，并在罪恶满溢期间承受以下不利影响。
- 妳需要标记3级伤害**精神崩溃**，在罪恶满溢解除时自动消除，这是治愈它的唯一办法。
- 妳承受的罪恶每超过罪阈1点，就需要标记1次1级伤害**发狂**。
- 妳陷入祢挑选的一种疯狂状态，也可以自创一个。

##### 罪行
罪行取代了原本的纵欲过度。从以下选项中选择妳犯下的罪行，GM与妳共同确定罪行在银屏上的具体表现。罪行带来的后果会立即生效，并可能影响后续的游戏进程。
- **侮蔑**：妳深深伤害了一段牵绊。妳可以选择：牵绊对象与妳断绝关系，妳无法再从她那里获得闪耀；或是面对一个纠葛。
- **狂想**：妳的妄想驱使妳做出极端行为，直到妳消耗1次休整活动从妄想中清醒。
- **沈默**：妳拒绝与世界沟通。直到下次休整期开始前，妳无法通过情感联结获得闪耀，并且使用命令、交结、动摇的行动骰-1D。
- **安息**：妳放弃了对某件事物的坚持。妳可以选择：从一项正在进行的长期项目进度钟上移除1格刻度；或是选择1项议程，无法在本场会话触发。
- **真理**：妳揭露了一个本应保守的秘密。集团罪熵+1。
- **贪食**：妳沉溺于享乐，挥霍无度。妳失去2存款或1硬币。
- **姦淫**：妳的欲望驱使妳玷污了某段牵绊或冒犯了某个派系。妳可以选择：失去1名声；或是与相应的派系或地区公民的关系-1。

#### 创伤
场景结束时，如果妳处于罪恶满溢，便清空罪恶，并标记1道创伤。那份疯狂不再是一时兴起的冲动，成为了妳持续性的驱动力。 \
每道创伤都会占用1点闪耀容量。一旦占用的容量超过［闪耀上限÷2］点，便意味着妳的心灵已经破败不堪。祢无法再扮演这名角色，将控制权交给GM。 \
在那之前，妳可以尝试从创伤当中恢复。与GM合作，来制定一个长期项目 (或其他方法) 从而消除妳的创伤。每个角色的创伤都是个性化的，康复的途径也应该各不相同。

> 这道创伤让妳想要什么？害怕什么？相信什么？康复的关键就是找到这些问题的答案。

##### 牵动创伤
祢在每场会话中都可以牵动1次创伤，为自己的角色制造麻烦或恶化处境。当祢如此做时，标记1点经验。 \
只有祢能够牵动自己角色的创伤，其他玩家都不行。

#### 闪耀
妳最多能够同时拥有6点闪耀，创伤会占用其中的容量。 \
妳随时[^note-闪耀1]可以清空2个提升进度钟，增加1点闪耀上限，并获得1点闪耀。通过这种方式，闪耀上限最多可以提升至12点。 \
在游戏过程中，妳还可以通过以下方式获得闪耀。
- **舔舐伤口**：如果闪耀低于2点，妳可以消耗1次休整活动，令闪耀在休整期结束时变为2点。
- **逼迫自我**：当妳为牵绊对象逼迫自我时，除了原本的增益，额外获得1点闪耀。
- **情感联结**：当妳演绎出与牵绊对象深刻的情感互动，或完成一个与牵绊相关的个人目标时，GM可以奖励妳1点闪耀[^note-闪耀2]。通过这种方式，每段牵绊在每场会话最多可以提供2点闪耀。 \
  如果妳的互动对象处于罪恶满溢，妳不会因此获得闪耀。

[^note-闪耀1]: 不需要训练。
[^note-闪耀2]: 妳的牵绊角色也会因此获得闪耀。

> 妳想要什么？妳愿意为此付出什么？这是情感互动的两个核心问题。

当祢在银屏上做出合适的扮演时，可以通过消耗闪耀发起以下行为。

:::{admonition} 重振精神
消耗任意点闪耀，消除等量的罪恶。
:::

:::{admonition} 本能反应
每场会话1次，消耗1点闪耀，在1次威胁骰中加入1个6。这不会使妳获得刀锋。
:::

:::{admonition} 偏差认知
消耗1点闪耀，重掷这次掷骰。可以重掷已经重掷过的掷骰。
:::

:::{admonition} 通晓万物
消耗1点闪耀，向GM提出一个问题，GM必须如实作答。 \
妳的问题需要满足以下所有条件。
1. **形式**：必须是一个可判定真假的命题。
2. **范围**：仅涉及已揭示或可直接推导的当前状况。
3. **原子性**：不得包含逻辑连接词 (例如"且"、"或"、"如果…那么…") ，不得嵌套提问。
4. **非元问题**：不得涉及游戏机制、未来发展、他人意图或道德判断。

GM根据命题$P$与当前世界状态$W$的关系，给出以下回应之一。
|   回应   |        判定条件         |
| :------: | :---------------------: |
| **肯定** |     $P$在$W$中为真      |
| **否定** |     $P$在$W$中为假      |
| **不定** | 真值无法由当前信息确定  |
| **无关** | $P$涉及未定义或无效内容 |
:::

:::{admonition} 闪回
闪回不再需要承受罪恶，而是消耗等量的闪耀。
:::

##### 可选规则：宠爱骰
当妳通过行动与牵绊对象进行情感上的互动时，可以向GM申请掷宠爱骰，从而获得0~3点闪耀。 \
妳所用的行动每有1点就掷1D，外加妳所拥有的奖励骰。妳无法对罪恶满溢的角色掷宠爱骰。
| 出目  |     效果      |
| :---: | :-----------: |
|  6,6  | 妳获得3点闪耀 |
|   6   | 妳获得2点闪耀 |
|  4~5  | 妳获得1点闪耀 |
|  1~3  | 妳获得0点闪耀 |

如果妳的互动对象是一名角色，当妳掷宠爱骰时，她也可以根据自己的行动，对妳掷宠爱骰。 \
根据GM的判断，妳也可以使用其他更合适的数值掷宠爱骰，比如属性、集团层级或生活水平。


## 亵渎
亵渎是使徒内心纷扰的精神显化，从使用者的潜意识中汲取力量，允许她突破凡人的极限。 \
通过玄奥仪式、火花工艺或是上天的眷顾，妳有机会获得亵渎的力量，成为一名使徒。 \
与GM合作决定妳是否拥有亵渎，以及它的来源。

拥有亵渎后，妳可以在休整期清空2个提升进度钟，获得1种新的亵渎。这不会消耗休整活动。 \
妳会从获得的首个亵渎掌握2道术式，再从之后获得的每种新亵渎掌握1道术式。 \
妳可以通过以下方式使用亵渎。
- 妳仅因拥有亵渎就能获得其被动增益。
- 妳可以承受罪恶来激活妳从亵渎掌握的1道特定术式。
- 妳总是可以承受2点罪恶展现**奇迹**，为自己或他人的1次掷骰+1D。
- 妳能够以亵渎作为起源学习并举行仪式，即使妳没有**仪式**特殊能力。 \
  获得**仪式**特殊能力后，妳举行术式之外、以亵渎作为起源的仪式所掷的行动骰+1D。

> 力量从不免费。问问祢的角色：妳愿意为了什么而堕落？如果必须牺牲一段牵绊来换取力量，妳会选哪个？

### 恩典
亵渎的量级由使徒的**恩典**等级所衡量。 \
当妳进行与亵渎相关的掷骰时，基础的骰子数量会变为原本的一半[^note-恩典]，再加上妳的恩典。这也包括妳在抵抗亵渎时的掷骰。

[^note-恩典]: 奖励骰不受影响。

> 随着恩典增长，祢的角色离凡人越来越远。她还能认得自己吗？她还在乎吗？

#### 启迪进度钟
如果妳尚未拥有亵渎，但通过某种方式获得了启迪，创建1个4格的进度钟，并将启迪填入其中。 \
当进度钟填满时，妳获得1种亵渎，并成为**使徒**。
- 如果妳此时没有任何创伤，需要标记1道创伤。
- 如果妳此时的谱系为**凡**，将会变为**魔**。

如果妳在角色创建完成时已经拥有亵渎，将会创建1个［2+起始恩典×4］格的进度钟。询问GM来确定妳的起始恩典等级。

每次填满启迪进度钟时，将妳的恩典提升1级，随后创建1个［2+当前恩典×4］格的进度钟。溢出的启迪会填充在新的进度钟上。

#### 获得启迪
会话结束时，妳在这场会话中每触发以下任一情形，可以获得1点启迪。若同一情形多次触发，改为获得2点。只有PC会因此获得启迪。
- **直面禁忌**：当妳主动踏入凡人不该涉足的领域——无论是阅读禁书、探访诅咒之地，还是与异界存在交谈——并且这段经历对妳造成了深刻的冲击或改变。
- **撕裂自我**：妳为了守护或伤害某段牵绊，做出了违背自身信念的行为。那一刻，妳听见内心有什么东西碎裂的声音。
- **承受极致**：妳的罪恶在同一场景从0达到罪阈，或是在罪恶满溢的状态下度过大半个刻痕并承受创伤。
- **牺牲闪耀**：妳消耗了3点或更多闪耀，用以达成某种极端目的。
- **遗留执念**：妳承受了4级伤害而未死。濒死体验让妳的感知穿透了现实的帷幕。
- **联结异类**：妳与一名使徒建立了深刻的情感牵绊，或是与她的亵渎产生了共鸣——例如，共享幻觉、同时梦见同一场景、或是在关键时刻心意相通。
- **见证奇迹**：仅限凡类。妳亲眼目睹了无法用常理解释的现象——一道不应存在的刻痕、一个死而复生的人、或是亵渎力量具现化的瞬间。妳的认知因此被动摇。
- **完成仪式**：仅限凡类。妳成功举行了一场与亵渎相关的仪式，即使妳尚未拥有亵渎。仪式的代价与成果让妳的潜意识开始觉醒。
- **破解谜题**：仅限凡类。妳解开了某个与亵渎直接相关的古老谜题、预言或封印。知识本身就是钥匙，而妳转动了它。
- **传承烙印**：仅限凡类。妳从一名垂死的使徒那里继承了某种印记、遗物或记忆。那烙印灼烧妳的灵魂，迫使妳踏上相同的道路。触发本情形获得的启迪提升至2D3点。

:::

:class: seealso

:::

:::{admonition} 心灵
:class: seealso
**主题**：沟通、语言、幻觉、思想、记忆、心灵投射

**被动**：当妳与某人进行一对一交谈时，妳能模糊感知对方当下的主导情绪 (愤怒、恐惧、欲望等)。每场会话1次，可以令妳或同伴利用此信息进行的行动骰+1D，或是改善处境・效果。

### 术式
术式是使徒的战斗本能——在追逃之间、在生死之际，从亵渎本质中榨取力量的简短仪式。 \
举行术式通常只会消耗极短的时间，因此无需消耗休整活动。

术式的强度分为**轻微**、**中等**、**强大**三个等级，分别需要妳在游戏过程中，曾经同时拥有过至少1道/2道/3道创伤才能掌握。

#### 创造术式
祢可以根据以下步骤为亵渎创造术式。

##### 一、选择系别
从六种系别中选择一种，作为术式的作用方式。系别决定了妳**如何**操控力量。

| 系别  | 描述                                                                                                 | 范例                                                                                   |
| :---: | :--------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------- |
| 统御  | 妳控制现象，使其按妳的意愿运作。让火焰避开妳的同伴，让锁芯在妳的命令下转动，让幽灵在妳面前停滞。     | **轻微**：让硬币翻转；**中等**：让水流改道；**强大**：让时间加速。                     |
| 探知  | 妳获取隐藏的知识或揭露真相。看见凶案现场的最后一幕，感知某人是否发下血誓，听见灵质电线中流淌的信息。 | **轻微**：知晓尸体的死因；**中等**：看见无线电波；**强大**：洞悉恶魔的本质。           |
| 护持  | 妳防御、隐藏或保护目标。在周身构筑冰墙，让妳在死者面前隐形，用灵质护盾抵挡枪弹。                     | **轻微**：隐藏一件小物品；**中等**：抵抗一种能量；**强大**：在幽灵领域中抹除自身存在。 |
| 毁灭  | 妳弱化、分解或彻底摧毁目标。让钢梁锈蚀，让敌人的意志崩溃，让两个地点之间的距离彻底消失。             | **轻微**：削弱敌人的力量；**中等**：侵蚀肉体；**强大**：抹除记忆或距离。               |
| 附魔  | 妳强化、修复或创造事物。让濒死的伤口愈合，让破碎的遗物复原，从虚无中召唤出短暂的火焰。               | **轻微**：修复小物件；**中等**：强化武器；**强大**：从灵质中创造实体。                 |
| 重塑  | 妳改变事物的性质或形态。将钢铁变得如橡胶般柔软，改写目标几秒内的记忆，将自身血肉暂时化为灵质蒸汽。   | **轻微**：改变物体颜色；**中等**：重塑面部特征；**强大**：转化为幽灵形态。             |

##### 二、定义效果
就像寻常的仪式那样，与GM进行有关仪式的问答，来确定术式的游戏效果，但它并不一定怪诞。 \
一道术式只能达成一个明确的目标。可以是「让火焰喷射从而烧伤敌人」，不能是「让火焰喷射烧伤敌人、同时加热武器、顺便吓跑老鼠」。 \
祢还需要为术式选择1种**首要量级** (面积、距离、威力等)，选择的简单原则是：提到术式的更强版本时，祢先想到哪一个量级。 \
术式在其首要量级上的表现，最高可以达到妳的恩典等级。其他方面的量级由GM根据直觉判断。 \
祢不能创造多道除了首要量级之外完全相同的术式。

##### 三、明确代价
激活术式所需承受的罪恶由其强度决定，与量级无关。
- **轻微**：产生1点罪恶。
- **中等**：产生1D3点罪恶。
- **强大**：产生2D3+点罪恶。骰子枚数由GM根据直觉判断。
- **附加条件**：祢可以与GM合作，设置一些额外的条件来减轻术式产生的罪恶，最低降至1点。 \
  例如，需要触碰、会在特定条件下降低效果、遭受伤害、花费硬币、失去物品、失去声名、推进进度钟、承担罪熵、失去派系关系。

##### 四、命名
为祢精心设计的术式取一个独一无二的名字。术式设计完成。

### 奇迹
这是亵渎最常见也最简单的运用方式。承受2点罪恶，在银屏上叙述妳如何展现奇迹。 \
妳可以通过奇迹为自己或他人的1次掷骰+1D。

妳也可以制造更具创造性的现象，来改变处境・效果，这需要使用同调掷行动骰[^note-奇迹]。 \
奇迹的量级初始为0级，妳可以将恩典分配在各个量级之上。 \
例如，恩典4级的使徒，可以将奇迹的威力提升至4级，或是将面积与持续时间各提升至2级。

[^note-奇迹]: 就像其他与亵渎相关的掷骰那样，将同调减半再加上妳的恩典，来决定基础的骰子数量。

除此之外，奇迹还可以达成以下简单用途。以这种方式展现奇迹不需要承受罪恶。
- **共感**：与相同场景的自愿使徒共享自己的感官。能够共享的感官仅限视觉、听觉、味觉。
- **念话**：与相同场景的自愿使徒进行意念交流。 \
  妳也可以消耗1次休整活动，与任意距离的自愿使徒进行一场简短的意念交流。
- **飞行**：消耗1次休整活动，全神贯注地快速飞行，能够飞到接近云层的高度，速度最高为普通人步行的15倍，直到停止飞行。在此期间，妳无法采取其他需要集中精神的行为。
- **变装**：消耗1次休整活动，用视觉上的幻象掩盖自己的外表，让自己在别人眼中变成另一种样子，直到下次休整期开始。无法把自己伪装成特定的角色。妳随时可以提前解除幻象。


## 罪种
一些幸运儿天生具备高浓度的恩典，这些神秘力量导致她们的大脑或是心脏处显现小型的肿瘤状赘生物，被称为罪种。 \
使用亵渎会在使徒内在的心灵之海中积累残余能量，这种能量由原始而强大的情感构成。如果没有宣泄口，能量便会溢出，侵蚀使用者的理智。 \
罪种的存在则能将这种能量转化为**悲怆**，帮助使徒中和汹涌的情感激流。

### 悲怆
妳拥有的悲怆等于当前创伤数量。 \
展现奇迹时，妳可以消耗1点悲怆代替承受罪恶。 \
激活术式时，妳可以消耗任意悲怆作为额外的附加条件，每点悲怆都能减少1D3点[^note-悲怆]罪恶，最低降至0点。 \
每次进入休整期时，妳会恢复所有被消耗的悲怆。

[^note-悲怆]: 对于中等・强大术式，将会直接减少骰子枚数。

### 罪印
超过界限的狂暴心能会在罪种的催化下溢出，为肉体留下明显的物理表现。 \
身具罪印的使徒命不久矣。她们染上了永久性的污点。

拥有罪种的使徒每次陷入罪恶满溢时，投掷［[罪印数量]D6］。掷出至少1个6则获得1个罪印。 \
每个罪印都会使妳的罪阈-1。参考下表决定罪印显化的外表。
| 部位  | 典型外表                                                                             |
| :---: | :----------------------------------------------------------------------------------- |
| 眼部  | 黑色或异色巩膜；白色虹膜；开裂状、狭长状或复数瞳孔；眼球表面浮现脉状纹理。           |
| 面部  | 下颚开裂或伸长；咽部存在咽颌；獠牙外露；异色舌头；黏性唾液；面部皮肤出现裂口或色块。 |
| 颈部  | 第二声带；可以扭曲至反常角度；皮肤下浮现管状结构或环状软骨增生。                     |
| 胸部  | 胸骨外凸或内陷；肋骨增生穿出体表；组织萎缩或异化为腺体结构；皮肤半透明可见脏器轮廓。 |
| 躯体  | 脊椎棘刺突出；皮肤变色、松弛或硬化；额外肋骨无序生长；退化的假翼从肩胛或肋侧伸出。   |
| 臂部  | 双臂失色；肌肉纤维化或扭曲增生；关节可以反向弯折；皮肤出现鳞片或板状硬皮。           |
| 手部  | 手指数量增减；指节间长有利爪；掌心开裂形成嘴巴或眼睛；手掌皮肤透明。                 |
| 腰部  | 腰椎侧弯或前突；皮肤出现环状勒痕或增生带；腰侧裂开露出内骨骼或副肢。                 |
| 脚部  | 足部反曲或蹄化；趾骨延长形成爪握结构；脚底分泌粘性体液；行走无声或留下腐蚀痕迹。     |

首次获得罪印时，创建1个4格的堕落进度钟。之后每次获得罪印，为堕落进度钟填充1格刻度。 \
填满堕落进度钟时，妳将被转化为一头可怖的惊骇憎物，成为受GM控制的NPC，并与所有玩家共同决定她接下来的命运。

#### 绽放罪印
妳可以绽放自己的罪印，达成以下超凡壮举。在银屏上描述妳是如何做到的。每个罪印只能用于达成一种壮举。
- 在极远距离之内清晰视物，仿佛能够“放大”视觉效果一般。
- 在短距之内透视墙壁和无生命物质。
- 在黑暗・雾气中清晰视物。
- 在短距之内以心灵感应的形式与其他生物交流。
- 在短距之内对单一目标进行吐息攻击，威力可达3级。
- 不再需要呼吸，也不会受到任何世俗毒素的影响。
- 在开阔区域，如路面上奔跑或冲刺时的行动骰+1D。
- 如果肉体无遮挡地触碰固体表面，妳可以在移动时将其视作平地。
- 承受1点罪恶，通过直视使一个凡类暂时瘫痪，持续至妳移开视线。
- 承受1点罪恶，对一个凡类发出不超过一个单词的简单命令。对方会立即尽其所能做到最好，并且不会服从任何显而易见地对自身有害的命令。
- 承受1点罪恶，展现惊人的体能，如同**不容小觑**特殊能力，量级可达3级。
- 承受1点罪恶，在空中滑翔一小段距离作为任意行动的一部分。
- 承受1点罪恶，重掷妳使用狩猎、研究或审视的行动骰。
- 承受1点罪恶，重掷妳使用灵巧、潜行或斗殴的行动骰。
- 承受1点罪恶，重掷妳使用命令、交结或动摇的行动骰。
