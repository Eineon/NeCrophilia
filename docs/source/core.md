# 追加规则

## 底层原则

- **最低值**：数值最低通常为0，不会被降至负值。
- **截尾取整**：计算结果出现小数时，通常会舍去小数的部分。例如：3.7取3，-1.2取-1。
- **特例优先**：**如果**某一项目与**通常**情况矛盾，**那么**优先应用特例的规则。
- **同伴**：除自身之外，GM认为与角色站在同一战线的其他角色。
- **幻化**：PL可以自由更改自身动作的名称与表现形式，以契合角色设定以及演出风格。此类修改不会影响动作在规则层面的实际效果。 \
  **如果**某项游戏项目基于动作的名称，**那么**始终以该动作幻化前的原始名称作为判定依据。
- **简易变量**：许多项目会以［N(编号)］的格式存储可变化的值。**如果**项目未指定变量的具体值，**那么**由角色在每次使用时自行决定变量代表的值。 \
  不同项目中的同编号变量不会相互引用。同时使用多个包含未知简易变量的动作时，角色需要分别决定各个动作的简易变量。 \
  简易变量存储的值可以是负值，并且会保留小数。

### 生物
所谓的**生物**是特指在机制上被定义为生物的实体，无关其是否具有生命。 \
例如，像机壳、亡灵这样无生命的实体，依然属于生物的范畴。随处可见的草木与细菌虽然拥有生命，但在机制上并不被视为生物，而是当作物品来对待。

所有生物都分属于以下谱系之一。
| 谱系  | 含义                                                             |
| :---: | :--------------------------------------------------------------- |
|  凡   | 具备心智的天然生命体。涵盖包括人类在内的所有自然动物。           |
|  异   | 脱离凡人行列的超自然存在，例如幽灵、吸血鬼与恶魔。               |
|  械   | 通过人工技术构建的生命体，通常缺乏情感与欲望，依赖预设逻辑运行。 |

### 掷骰

#### 奖励骰
妳可以在掷骰后，GM宣布后果前，决定是否为这次掷骰赋予那些妳可以主动添加的奖励骰。 \
无论如何，每次掷骰最多只能获得3枚奖励骰。

#### 重掷
当妳进行重掷时，重新投掷每枚D6，并使用新的出目。每枚骰子通常只能重掷1次。奖励骰不会被重掷影响。


## 牵绊
这一模块取代了核心《暗夜刀锋》中的亲朋、宿敌、恶欲，并调整了压力与创伤的机制。

### 建立牵绊
祢随时可以向GM申请为妳与特定对象建立牵绊，或是主动遗忘一段牵绊。 \
牵绊的对象可以是角色、组织、地域、甚至理念。但不能是妳自己。 \
妳也不能重复对相同对象建立多段牵绊。即便牵绊对象不复存在，已经建立的牵绊也不会消失。

### 临时牵绊
有时妳会被授予临时的牵绊，代表妳与特定对象的短暂关系。妳通常会在刻痕结束时遗忘所有临时牵绊。 \
祢随时可以向GM申请将临时牵绊转正。

### 情感
妳对待牵绊的情感分为2个侧面：**闪耀**与**罪恶**。 \
使妳标记压力或枯竭的机制改为给予等量的罪恶。 \
妳的**罪阈**决定了妳能承受的罪恶上限，初始为9。

#### 罪恶满溢
罪恶一旦到达罪阈，妳的心智便会被翻涌的恶意压垮。将罪恶满溢作为一个3级伤害事实，会在罪恶低于罪阈时自动消除，这是治愈它的唯一办法。 \

#### 创伤
刻痕结束时，如果妳处于罪恶满溢，便标记1道创伤。那份疯狂不再是一时兴起的冲动，成为了妳持续性的驱动力。 \
每道创伤都会占用2格闪耀上限。一旦占用了所有闪耀，便意味着妳的心灵已经破败不堪，失去了自我。祢无法再扮演这名角色，将控制权交给GM。 \
在那之前，妳可以尝试从创伤当中恢复。与GM合作，来制定一个长期项目 (或其他方法) 从而消除妳的创伤。每个角色的创伤都是个性化的，康复的途径也应该各不相同。

##### 牵动创伤
祢在每场会话中都可以牵动1次创伤，为自己的角色制造麻烦或恶化处境。当祢如此做时，标记1点经验。 \
只有祢能够牵动自己角色的创伤，其他玩家都不行。

#### 闪耀
妳最多能够同时拥有6点闪耀，创伤会降低这一上限。妳可以通过以下方式获得闪耀。
- **逼迫自我**：为牵绊对象逼迫自我时，除了原本的增益，妳还会获得1点闪耀。
- **情感联结**：当妳演绎出与牵绊对象深刻的情感互动，或完成一个与牵绊相关的重要个人目标时，GM可以奖励妳们每人【hint:如果妳的牵绊对象是角色】1点闪耀。同一牵绊在每场会话最多提供2点闪耀。 \
  如果妳的互动对象罪恶满溢，妳不会因此获得闪耀。

当祢在银屏上做出合适的扮演时，可以通过消耗闪耀发起以下行为。

:::{admonition} 重振精神
:name: 重振精神
消耗1点闪耀，消除1点罪恶。
:::

:::{admonition} 本能反应
:name: 本能反应
每场会话1次，消耗1点闪耀，在1次威胁骰中加入1个6。这不会使妳获得刀锋。
:::

:::{admonition} 偏差认知
:name: 偏差认知
消耗1点闪耀，重掷这次掷骰。可以重掷已经重掷过的掷骰。
:::

:::{admonition} 通晓万物
:name: 通晓万物
消耗1点闪耀，向GM提出一个问题，GM必须如实作答。 \
妳的问题须满足以下所有条件。
1. **形式**：必须是一个可判定真假的命题。
2. **范围**：仅涉及已揭示或可直接推导的当前状况。
3. **原子性**：不得包含逻辑连接词 (例如"且"、"或"、"如果…那么…") ，不得嵌套提问。
4. **非元问题**：不得涉及游戏机制、未来发展、他人意图或道德判断。

GM根据命题$P$ 与当前世界状态$W$ 的关系，给出以下回应之一。
|   回应   |         判定条件         |
| :------: | :----------------------: |
| **肯定** |     $P$ 在$W$ 中为真     |
| **否定** |     $P$ 在$W$ 中为假     |
| **不定** |  真值无法由当前信息确定  |
| **无关** | $P$ 涉及未定义或无效内容 |
:::

##### 可选规则：宠爱骰
当妳通过行动与牵绊对象进行情感上的互动时，可以向GM申请掷宠爱骰，从而获得0~3点闪耀。 \
妳所用的行动每有1点就掷1D，外加妳所拥有的奖励骰。妳无法对罪恶满溢的角色掷宠爱骰。
| 出目  |     效果      |
| :---: | :-----------: |
|  6,6  | 妳获得3点闪耀 |
|   6   | 妳获得2点闪耀 |
|  4~5  | 妳获得1点闪耀 |
|  1~3  | 妳获得0点闪耀 |

如果妳的互动对象是一名角色，当妳掷宠爱骰时，她也可以根据自己的行动，对妳掷宠爱骰。 \
在某些特殊情况下，妳可以使用其他更合适的数值掷宠爱骰，比如属性、集团层级或生活水平。


## 亵渎
亵渎是使徒内心纷扰的精神显化，从使用者的潜意识中汲取力量，允许使用者突破凡人的极限。 \
通过玄奥仪式或是禁忌的火花工艺，妳可以获得亵渎的力量。这会使妳成为一名使徒，凡系生物的谱系将因此变为**异**。

妳会从获得的首个亵渎掌握2道术式，再从之后获得的每种新亵渎掌握1道术式。 \
妳可以通过以下方式使用亵渎。
- 妳仅因拥有亵渎就能获得其被动增益。
- 妳可以承受罪恶来激活妳从亵渎掌握的1道特定术式。
- 妳随时可以承受2点罪恶，为自己或他人的1次掷骰+1D，在银屏上叙述妳如何创造性地使用亵渎。
- 妳能够以亵渎为中心开发与举行仪式，即使妳没有**仪式**特殊能力。妳为举行这样的仪式而承受的罪恶，减少等于妳恩典的点数，最低2点。 \
  如果有**仪式**特殊能力，在举行这样的仪式时，妳的恩典+2。

### 恩典
亵渎的量级由使徒的**恩典**等级所衡量。没有亵渎的角色无法拥有恩典等级。 \
当妳进行与亵渎相关的掷骰时，将基础的骰子枚数折半【hint:奖励骰不会折半】，随后每级恩典+1D。这也包括妳在抵抗亵渎时的掷骰。

### 启迪进度钟
如果妳尚未拥有亵渎，但通过某种方式获得了启迪，创建1个4格的进度钟，并将启迪填入其中。 \
当进度钟填满时，妳获得1种亵渎，成为一名**使徒**。如果妳此时没有任何创伤，需要标记1道创伤。

如果妳在角色创建完成时已经拥有亵渎，将会创建1个［2+起始恩典×4］格的进度钟。询问GM来确定妳的起始恩典等级。

每次填满启迪进度钟时，将妳的恩典提升1级，随后创建1个［2+当前恩典×4］格的进度钟。溢出的启迪会填充在新的进度钟上。

任何未明确提及启迪进度钟的机制都无法影响它的进度。

#### 获得启迪
会话结束时，妳在这场会话中每触发以下任一情形，便获得1点启迪。若同一情形多次触发，改为获得2点。
- **直面禁忌**：当妳主动踏入凡人不该涉足的领域——无论是阅读禁书、探访诅咒之地，还是与异界存在交谈——并且这段经历对妳造成了深刻的冲击或改变。
- **撕裂自我**：妳为了守护或伤害某段牵绊，做出了违背自身信念的行为。那一刻，妳听见内心有什么东西碎裂的声音。
- **承受极致**：妳的罪恶在一瞬间从0达到罪阈，或是在罪恶满溢的状态下度过了整个刻痕。
- **牺牲闪耀**：妳在这场会话中消耗了3点或更多闪耀，用以达成某种极端目的。
- **遗留执念**：妳承受了4级伤害而未死。濒死体验让妳的感知穿透了现实的帷幕。
- **联结异类**：妳与一名使徒建立了深刻的情感牵绊，或是与她的亵渎产生了共鸣——例如，共享幻觉、同时梦见同一场景、或是在关键时刻心意相通。
- **见证奇迹**：仅限凡系生物。妳亲眼目睹了无法用常理解释的现象——一道不应存在的刻痕、一个死而复生的人、或是亵渎力量具现化的瞬间。妳的认知因此被动摇。
- **完成仪式**：仅限凡系生物。妳成功举行了一场与亵渎相关的仪式，即使妳尚未拥有亵渎。仪式的代价与成果让妳的潜意识开始觉醒。
- **破解谜题**：仅限凡系生物。妳解开了某个与亵渎直接相关的古老谜题、预言或封印。知识本身就是钥匙，而妳转动了它。
- **传承烙印**：仅限凡系生物。妳从一名垂死的使徒那里继承了某种印记、遗物或记忆。那烙印灼烧妳的灵魂，迫使妳踏上相同的道路。触发本情形获得的启迪提升至2D3点。

### 术式
术式是使徒在战斗中、在追逃间、在生死一线的抉择时刻，从亵渎的本质中榨取的力量迸发。它不需要祭坛、不需要准备、不需要晦涩的咒文——只需要妳愿意承受代价。 \
作为一个只会清空2个提升进度钟的资深能力，妳可以从当前拥有的所有亵渎中掌握合计2道术式。

#### 创造术式
当妳获得亵渎时 (首次获得掌握2道术式，后续每种新亵渎掌握1道)，祢可以根据以下步骤为该亵渎创造术式。

##### 一、选择形态
从六种形态中选择一种，作为术式的作用方式。形态决定了妳**如何**操控力量。

| 形态  | 描述                                                                                                 | 范例                                                                                   |
| :---: | :--------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------- |
| 统御  | 妳控制现象，使其按妳的意愿运作。让火焰避开妳的同伴，让锁芯在妳的命令下转动，让幽灵在妳面前停滞。     | **轻微**：让硬币翻转；**中等**：让水流改道；**强大**：短暂加速自身时间流。             |
| 探知  | 妳获取隐藏的知识或揭露真相。看见凶案现场的最后一幕，感知某人是否发下血誓，听见灵质电线中流淌的信息。 | **轻微**：知晓尸体的死因；**中等**：看见无线电波；**强大**：洞悉恶魔的本质。           |
| 护持  | 妳防御、隐藏或保护目标。在周身构筑寒冰屏障，让妳在死者面前隐形，用灵质护盾抵挡枪弹。                 | **轻微**：隐藏一件小物品；**中等**：免疫一种能量；**强大**：在幽灵领域中抹除自身存在。 |
| 毁灭  | 妳弱化、分解或彻底摧毁目标。让钢梁锈蚀，让敌人的意志崩溃，让两个地点之间的距离彻底消失。             | **轻微**：削弱敌人的力量；**中等**：侵蚀险棘；**强大**：抹除一段记忆或距离。           |
| 附魔  | 妳强化、修复或创造事物。让濒死的伤口愈合，让破碎的遗物复原，从虚无中召唤出短暂的火焰。               | **轻微**：修复小物件；**中等**：强化武器；**强大**：从灵质中创造短暂实体。             |
| 重塑  | 妳改变事物的性质或形态。将钢铁变得如橡胶般柔软，改写目标几秒内的记忆，将自身血肉暂时化为灵质蒸汽。   | **轻微**：改变物体颜色；**中等**：重塑面部特征；**强大**：短时间转化为幽灵形态。       |

##### 二、定义效果
术式的效果必须满足以下全部条件。
- **即时性**：效果必须在激活后立即显现，持续不超过一次刻痕。不能有「持续一天」「永久改变」之类的长效设计——那是仪式该做的事。
- **局部性**：效果范围不得超过「目视可及」或「一箭之地」(约50米)。不能影响整个街区、整座建筑——那是仪式该做的事。
- **单一性**：术式只能达成一个明确的目标。「让火焰喷射并烧伤敌人」可以，「让火焰喷射、烧伤敌人、同时照亮房间、顺便吓跑老鼠」不行。

##### 三、设定代价
激活术式所需承受的罪恶由效果强度决定：轻微效果1点，中等效果2点，强大效果3+点。
- **可选附加条件**：每项可以减轻1点产生的罪恶，最低1点。
  + 需要触摸目标
  + 需要在特定环境使用 (例如夜晚、雨中、墓地、闪电屏障附近)
  + 需要承受1级伤害作为副作用
  + 会在特定条件下降低效果
  + 引起敌对势力注意 (例如推进「灵魂监守聚焦」进度钟)

##### 四、命名
为祢精心设计的术式取一个独一无二的名字。祢可以自由设定其表现形式以贴合角色设定。

#### 更换术式
每次提升恩典等级时，妳都可以更换任意道术式。 \
妳也可以在休整期消耗1次休整活动以更换最多2道术式。
